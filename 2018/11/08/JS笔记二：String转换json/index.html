JS笔记二：String转换json

Posted on 2018-11-08 | In JS
通过popup使用ajaxFileUpload上传，遇到点问题，做个笔记。（注：ajaxFileUpload公司做了改造，不适用全部使用，基本处理方式差不多）

问题一：方法调用到了，但是获取不到返回数据

原因：找了些网上的资料，说是由于传了自定义参数，dataType不能使用json，但是其他的类似功能使用都可以，不过dataType改成post的确是可以获取参数。

问题二：通过data.res_code的方式获取返回码错误

原因：post返回的数据是String类型的，而data.res_code是json类型参数，所以失败。

问题三：转换json错误

使用json = eval(‘(‘ + json + ‘)’);方法转换String到json报错

最后发现是由于返回结果data数据有点问题：{“res_code”:”1300”,”res_message”:”查询不到相关数据！ \n上传成功条数：6，上传失败条数：0 \n 有效号码条数：0”,”result”:null}


最后通过substring截取可以转换成json的部分数据：var length = data.indexOf(“<”); var json = data.substring(0,length);

之后进行json = eval(‘(‘ + json + ‘)’);转换就正常了，就可以获取返回数据了。
