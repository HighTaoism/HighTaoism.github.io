oracle笔记三：导出导入数据

Posted on 2018-06-22 | In oracle
由于项目有2个数据库，一个数据库做日志，一个数据库做其他数据。然后需要对日志数据库进行分析，统计成视图给客户观看。因此需要把日志数据库关联另一个数据库的集团和组织等表来进行统计数据，最后把数据库的表数据同步到另一个日志数据库中。

导出数据

本次导出数据是通过shell脚本来执行。首先创建一个数据库目录，然后将数据dmp文件导入这个目录中,将数据scp到另一个服务器上。另外数据只保留最近三天

create directory iotlogdir as ‘/home/oracle/230to209’;
